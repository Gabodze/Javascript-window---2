Window ობიექტი

Window ობიექტი წარმოადგენს გახსნილ ფანჯარას ბრაუზერში

Window ობიექტის მეთოდები:

1. window.addEventListener - ფანჯარაზე ამაგრებს ივენთის დამმუშავებელს.

2. window.alert მეთოდი გამოიღენება მაშინ, როდესაც გვინდა რომ მომხმარებელს რაიმე ინფორმაცია მივაწოდოთ. შეტყობინება გამოდის პატარა ყუთის–ფანჯრის სახით ეკრანზე.

3. window.open-ში ხდება ვებგვერდების მისამართების გაწერა და იხსნება ახალ ჩანართში.

4. window.prompt-ით ხდება გარკვეული მისამართების შეტანა. რომელიც შემდეგ გამოდის ჩვენს ეკრანზე.
